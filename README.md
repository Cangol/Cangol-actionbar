Cangol-actionbar
---

An android library,custom simple actionbar

###Usage Maven

     <dependency>
         <groupId>mobi.cangol.mobile</groupId>
         <artifactId>actionbar</artifactId>
         <version>1.0.0</version>
         <type>aar</type>
     </dependency>
###Usage Gradle
 
    compile 'mobi.cangol.mobile:actionbar:1.0.0@aar'
