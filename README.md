### Usage


Maven

     <dependency>
         <groupId>mobi.cangol.mobile</groupId>
         <artifactId>actionbar</artifactId>
         <version>1.0.22</version>
         <type>aar</type>
     </dependency>
Gradle
 
    compile 'mobi.cangol.mobile:actionbar:1.0.22@aar'
